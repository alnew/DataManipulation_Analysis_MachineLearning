# DataManipulation_Analysis_MachineLearning
Homework and Final Project for Data Manipulation and Analysis course at the University of Michigan

## Homework Assignments Included:
- Homework 1 - Data Manipulation and analysis of historical data from the Olympic games

- Homework 2 - Univariate Statistics using higher education data

- Homework 3 - Visualization, Correlation, and Linear Models using YouTube and Pokemon data

- Homework 4 - Natural Language Processing with Project Gutenberg data
  * Word2Vec, chapter similarity, word clustering

- Homework 5 - Clustering by music preference
  * kMeans, hierarchical clustering
  * image compression using vector quantization 

- Homework 6 - Classification of malicious vs. bening websites
  * Random Forest (using feature importance)
  * Naive Bayes
  
- Homework 7 - Dimension Reduction using gene expression data
  * MDS
  * PCA
